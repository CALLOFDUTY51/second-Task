This is a simple e-commerce application 
that integrates with Stripe for payments. 
The app allows users to browse products, 
add them to a cart, and proceed to checkout via Stripe



Backend Setup

Navigate to the backend directory:

cd backend

Install dependencies:

npm install

Start the backend server:

nodemon



Frontend Setup

Navigate to the frontend directory:

cd frontend


Install dependencies:

npm install

Start the frontend development server:

npm run dev
